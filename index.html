Fast Food Till

Language: VB.NET
IDE: Visual Studio
This was the first big piece of software I developed, it's a till system developed for my A2 computing project. Based off of real till I developed this till so that all the buttons were customizeable and had different functions. In order to do this all the menu buttons are generated dynamically and all their information was stored in text files, perhaps not the safest of methods but it was how we were taught at school. The system also held totals for hourly checks and a daily checks as well as a stock control system and a staff hours tracking system which produced a bar chart based on how much a member of staff had worked.
